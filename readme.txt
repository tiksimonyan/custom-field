== Description ==

Custom Fields Display is a WordPress plugin that allows users to add and display custom fields on the front end of WordPress posts. It supports custom text, date, and image fields and provides user-friendly customization options.

== Installation ==

1. Upload the `custom-field` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

Once the plugin is activated, you can add custom fields to your posts in the WordPress post editor. The custom fields will be displayed on the front end of your website along with the post content.

To add custom fields:
1. Go to the post editor in WordPress.
2. Scroll down to the custom fields meta box.
3. Add your custom fields (text, date, image) as needed.
4. Update or publish the post.

To customize the display of custom fields on the front end:
- You can modify the CSS styles in the `css/style.css` file to match your website's design.
- You can customize the output of custom fields in the `display_custom_fields` function in the main plugin file (`custom-field.php`).
